# Партнерська програма

Демо-проект із трьох сторінок про роботу з партнерами. Підготовлено для розгортання на GitHub Pages.

## Структура

```
.
├── index.html          # Головна сторінка (індекс)
├── v1/
│   └── index.html      # Версія 1 — Базова партнерська програма
└── v2/
    └── index.html      # Версія 2 — Розширена партнерська програма
```

## Сторінки

- **/** — головна сторінка з оглядом програми та переліком переваг
- **/v1/** — базова версія: фіксована комісія 15%, разові виплати
- **/v2/** — розширена версія: багаторівнева система 20–30%, рекурентні платежі

## Як розгорнути на GitHub Pages

1. Створіть новий репозиторій на GitHub.
2. Завантажте всі файли (зберігаючи структуру тек) у репозиторій.
3. Перейдіть у `Settings → Pages`.
4. У розділі `Source` оберіть гілку (наприклад, `main`) та теку `/ (root)`.
5. Збережіть. Через кілька хвилин сайт буде доступний за адресою:
   - `https://<username>.github.io/<repo-name>/`
   - `https://<username>.github.io/<repo-name>/v1/`
   - `https://<username>.github.io/<repo-name>/v2/`

## Локальний перегляд

Просто відкрийте `index.html` у браузері або запустіть локальний сервер:

```bash
python3 -m http.server 8000
```

Потім відкрийте http://localhost:8000
